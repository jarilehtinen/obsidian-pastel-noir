# Pastel Noir

An Obsidian theme with pastel accents — supports both dark and light base color schemes.

![Pastel Noir screenshot](screenshot.jpg)

## Features

- **Dark and light modes** — palette and backgrounds tuned per mode; structural rules shared
- **Unified background** — near-black base in dark mode, near-white in light, across all UI areas
- **Pastel folder colors** — top-level and second-level folders get unique colors, deeper levels stay gray
- **Colored breadcrumbs** — path bar matches folder color hierarchy
- **Pastel headings** — each heading level has its own color
- **Flat tabs** — no rounding, minimal borders, clean look
- **Centered sidebar buttons** — Files, Search, Bookmarks centered in their container
- **Brighter icons and muted text** — improved readability over default themes
